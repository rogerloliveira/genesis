# GENESIS

A model project with some pre-configured features like Devise. 
Also includes an extensible front-end template with Dart-SaSS support.

## Requirements
* Ruby 3.1.2 
* Rails 7.0.4

## Included Gems
The following gems are preconfigured 
* Devise (Authentication)
* Kaminari (Pagination)
* Sidekiq (Job Manager)
* PG (PostgreSQL)

## JS Components
* image-preview
* text-box
* toggle-button
* expandable-list
